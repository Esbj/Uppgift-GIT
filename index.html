1. Jag loggade in på mitt GitHub konto (https://github.com/Esbj), navigerade till repositories och klickade på “new” och fyllde i uppgifter om namn och beskrivning.
  2. När repot var skapat skapade jag en mapp för kursen med git bash terminalen för windows och laddade ner repot med kommandot git clone.
  3. Efter det skapade jag en index.html i repot med komandot touch. Jag fyllde på dokumentet medd denna listan och gjorde en kommit via VS code source controll. Skrev lite mer och gjorde en kommit till innan jag tryckte på sync knappen i vs code eller körde GIT PUSH
  4. Jag gjorde en branch med kommandot git checkout -b "branch". Där ändrade jag min ul tagg till en ol tagg för att få siffror, samt skrev detta meddelande. Jag commitade denna gång med komandot git commit -am "message", bytte sedan branch med git swithc main och mergade sedan med komandot git merge branch, för att få in ändringarna i min huvudbranch.
  5. Jag pushade sedan upp mina ändringar med git push, efter att jag commitat denna raden
